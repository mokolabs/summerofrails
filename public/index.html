Got plans this summer? Good… because you're going to learn Ruby on Rails.

You installed Ruby on Rails, you bought the books, and you even made a shopping cart.

But that was months ago and you still haven't shipped your first Rails app.

So let's fix that.

Last Update: Summer of Rails update

Here's the Summer of Rails challenge:

1. Learn Rails

Read the books. Find out how to do something. Borrow some code snippets. Download gems. Try some test-driven development. There's a ton of stuff to learn.

2. Make something useful

You know, something your mom would use. Ok, well, it doesn't have to be that useful.

But here are a few ideas to get you started…

  • A single-page web app
  • A Rails version of an existing site
  • A Web 2.0 mashup

3. Badge it

When your project launches, badge it as a Summer of Rails project and add it to the list of projects.
